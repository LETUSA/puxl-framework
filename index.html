Puxl (ˈpʌksl)

Build the accessible Web.
Save energy.

Puxl is an accessibility-first, eco-friendly open source front-end library for the modern Web. It helps you create accessible, responsive Web User Interfaces and lightweight HTML.

  • Live accessibility errors detection on browser: elements are marked with the solution ready to copy-paste-solve.
  • Automatic color calculation for the perfect shade of your colors to pass contrast accessibility tests.
  • Simple, short class names to build intuitively with less HTML and divitis.
Documentation Download v0.1-βeta

Features

Accessibility error prevention

Fully customizable (even the look and shape of select and checkbox), is perfect for heavy html loading sites. As you build your interfaces, it guides and prevents you with common accessibility requirements.
Puxl CSS detects accessibility errors and displays them in browser on development time, describing the problem and providing a solution in a copy-paste-solve fashion.

Easy to use, saves energy

CSS class names use industry conventions so you can start using Puxl as quick as 1-2-3. We got rid of long CSS names and unnecessary node nestings that overpopulate HTML.

The less HTML, the less data to serve, and lower data and energy bills for your pocket and our mother Earth.

1-click color contrast

Color contrast won't be a problem anymore for Designers with built-in color algorithms. Load your desired colors in the theme palette, save and let Puxl do the heavy work in buttons, background-colors, colors, border-colors... all over your site.

Latest Web technologies, no worries

CSS custom properties (a.k.a. CSS variables) are here to stay and get along really well with JavaScript, we couldn't leave them out the party. Also there is built-in support for LTR - RTL, Puxl icons, and more.

Modular and extensible

Built with Sass, Puxl's modular and extensible architecture uses just the right code you need for your Webs. To do so, we introduce _setup.scss, the place where to start configuring your next project's needs.
